# Release Notes for Play Console

## Version 1.0.0

### Новые возможности
- Полная регистрация с обязательными полями (Имя, Фамилия, Роль)
- Улучшенная навигация с обработкой кнопки "Назад"
- Лента постов только от подписок
- Исправлены ошибки permission denied в чатах
- Улучшенное отображение имени и username

### Исправления
- Исправлена работа чатов
- Исправлена навигация
- Исправлена главная страница
- Улучшена производительность

### Технические улучшения
- Обновлены Firestore Rules и Indexes
- Добавлено логирование для автотестов
- Оптимизированы запросы к базе данных
