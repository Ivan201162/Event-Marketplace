# Changelog v5.22-mega-universe

## Версия 5.22.0 (Build 32)

### Новые функции

#### Лента Feed 2.0
- Лента показывает только посты/рилсы/сторисы от подписок
- Добавлены рекомендации (top-10 по активности)
- Поддержка лайков, сохранений, комментариев, репостов
- Пагинация (lazy load)
- Переход в full-view для постов/рилсов
- Сторисы автоматически удаляются через 24 часа

#### Социальная логика
- Система лайков для постов, рилсов, идей
- Сохранение контента (bookmarks)
- Подписки/подписчики с счётчиками
- Кнопки подписки/отписки в профиле

#### Комментарии
- Общий UI-компонент для комментариев
- Поддержка @упоминаний (базовая)
- Счётчик комментариев

#### Чаты 2.1
- Реакции на сообщения
- Закрепление сообщений
- Индикатор "печатает..."
- Счётчик непрочитанных
- Поддержка вложений

#### Календарь / Бронирования
- Исправлены баги с загрузкой
- Фильтр по времени (базовая структура)
- Улучшенные логи бронирований

### Улучшения

#### Firebase / Backend
- Добавлен offline cache для Firestore
- Обновлены правила безопасности
- Обновлены индексы
- Cloud Function для автоматической очистки сторис

#### Авторизация
- Улучшен Google Sign-In с auto-retry
- Fallback на refresh при ошибках

#### Аналитика
- Новые события: feed_opened, story_view, post_like, follow_user, booking_sent

### Исправления
- Исправлены баги с мини-окном загрузки в календаре
- Улучшена обработка ошибок авторизации
- Оптимизирована загрузка контента в ленте

### Технические детали
- Версия: 5.22.0+32
- BUILD_VERSION: v5.22-mega-universe
- Минимальная версия SDK: Android 8.0
- Размер APK: ~79.4MB
